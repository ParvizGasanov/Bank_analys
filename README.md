# Bank_analys

Detta repo innehåller min kunskapskontroll i Statistik och dataanalys.

## Innehåll
- `bankanalys.ipynb` – exekverad Jupyter Notebook med analys och klustring
- `Kunskapskontroll2.pbix` – Power BI-dashboard

## Kort sammanfattning
Analysen delar in bankens kunder i tre segment:
 Högspenderande aktiva kunder
 Lågaktiva vardagskunder
 Kontantuttagskunder

Syftet är att hjälpa banken att anpassa erbjudanden, marknadsföring och kreditstrategi efter olika kundtyper.
